# Agenda Front-end 📒

## 🛠️ Tecnologias

**Frontend:**

- Angular cli@19.1.7
- HTML
- CSS
- TypeScript

---

## ⚙️ Pré-requisitos

- npm 10.2.4
- node v18.19.1

---

## 📦 Instalação

```bash
# Instale as dependências e construa o projeto
npm install

# Execute a aplicação
ng serve

# A aplicação irá abrir na porta http://localhost:4200
```
